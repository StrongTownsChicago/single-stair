Chicago Building Code Reform

What If Chicago Allowed
Single Stair Buildings?

Above the second story, Chicago's building code requires each unit to have access to two stairways. On a standard 25-foot lot, that means three stairways per building, resulting in smaller, darker apartments with fewer bedrooms. Seattle, New York, Austin, and Honolulu already allow single stair with sprinklers, and more states are following.
